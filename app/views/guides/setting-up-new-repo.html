After cloning the master repo, you can create a new repo for the clone that's unique.

1. Go to your GitHub profile

Open GitHub and click on the profile icon in the top right hand corner.

2. Go to your repositories

Click the repositories tab.

3. Create a new repo

Click the green ‘New’ button.

4. Name the repository, set privacy levels and then create

Click create repository after doing this.

5. Go to your terminal

Run this command:

git remote rm origin

This will remove the connection to the previous repository you pushed to.

Then, run this command:

git remote add origin URL

Replace 'URL' with the URL of your clone. The URL of your new repository will be available to you to copy on the GitHub page you have just created.

For example: git remote add origin https://github.com/accountname/name-of-repo.git

Then, run this command:

git add .

Then, run this command and add a description of the repo:

git commit –m description here

For example: git commit –m cloned provider prototype

5. Push the code

Run this command:

git push
