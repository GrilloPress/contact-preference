1. Push your code from the master repo (contact-preference) so it's up to date

Commit and push your code using the git tab in your text editor.

2. Go to or open your cloned repo and pull the latest code

If you named your clone remote-lab for example, open this folder in your text editor.

Fetch and then pull the new code from the master repo.

Do not use the git tab to push the changes! We'll use the terminal to do this to push only to Heroku.

3. Make sure you're in the right folder

If you named your clone remote-lab for example, use the terminal to navigate to that folder. To do this, run:

cd remote-lab

4. Push the code to Heroku

Run this command:

git push heroku

This might take a few minutes to build.

5. Check it's working

Run this command:

heroku open
