1. Make sure you're in the right folder

Use the terminal to navigate to the folder where your prototypes are. To do this, run:

cd name of folder

2. Clone the repo

Think of a name for the clone, for example, remote-lab. For this example, you would enter this command:

git clone https://github.com/GrilloPress/contact-preference.git remote-lab

Just change the name when you do this.

3. Navigate to your new folder

For example, if your clone name was remote-lab you would enter this command:

cd remote-lab

4. Install

Enter this command:

npm install

5. Check it's working

Enter this command:

npm run watch
