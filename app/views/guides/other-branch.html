1. Make sure you're in the right folder

If your prototype is called remote-lab for example, use the terminal to navigate to that folder. To do this, run:

cd remote-lab

2. Pull the latest

Run this command:

git pull

3. Switch to the branch

Run this command:

git checkout branch-name

Change branch-name to the name of the branch you want to work from.

You're now working from a new branch.
