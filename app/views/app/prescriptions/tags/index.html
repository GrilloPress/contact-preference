How many prescription tags there are

There are 3 prescriptions tags:

  • Requested
  • Approved
  • Rejected

The NHS service manual sass for tags is on Github.

Prescriptions tags we use
Class name Tag
nhsuk-tag--orange Requested
nhsuk-tag--green Approved
nhsuk-tag--red Rejected

Which users will see prescriptions tags

Users who will see prescriptions tags are:

  • EMIS users
  • Microtest users
  • Vision users

Where prescriptions tags appear

Prescriptions tags appear on the Your orders page.
